# SecurePlan AI Non-Technical Documentation

## What Is SecurePlan AI?

SecurePlan AI is a tool that helps teams create security testing documents for a website or web application.

Instead of manually writing a security test plan from scratch, the user enters the website URL. The system studies the visible pages and creates a professional security test plan and test scenarios.

## Why Is This Useful?

Security testing requires planning. Testers need to know what to check, which risks matter, and how to document their work clearly.

SecurePlan AI helps by:

- Saving time.
- Creating consistent security test plans.
- Covering important web security areas.
- Producing documents in Markdown, text, and JSON.
- Helping technical and non-technical stakeholders understand the test scope.

## How A User Uses It

1. The user opens the SecurePlan AI dashboard.
2. The user enters the landing URL of a web application.
3. The user chooses how many pages to analyze.
4. The user starts the analysis.
5. The system discovers pages, forms, buttons, and possible user flows.
6. The system maps those findings to security risk areas.
7. The system generates a security test plan.
8. The user exports the result.

## Example

If the website has a login page, SecurePlan AI may suggest tests for:

- Invalid login attempts.
- Password handling.
- Account lockout.
- Session logout.
- Access to protected pages.

If the website has a search box, SecurePlan AI may suggest tests for:

- Unsafe input.
- Script-like text.
- Error messages.
- Data leakage.

## What It Does Not Do

SecurePlan AI does not hack a website. It creates a plan and scenarios for authorized security testing.

The tool should only be used on websites the user owns or has permission to test.

## Business Benefit

SecurePlan AI helps teams move faster from application discovery to security test planning. It improves consistency, reduces manual effort, and creates professional documentation suitable for reviews, audits, and assessment presentations.
